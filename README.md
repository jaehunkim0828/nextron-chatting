## How to start

1. git clone
```
$ git clone https://github.com/jaehunkim0828/nextron-chatting.git
```
2. npm or yarn
```
$ npm i
$ yarn
```

3. build
  - package.json 참고
```
$ yarn build:all
```
4. dist
  - dist 파일에서 프로젝트를 찾으실 수 있습니다.
